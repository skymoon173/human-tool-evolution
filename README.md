# 人类工具进化史 🌍

> 从石头到戴森球 · 跨学科合成游戏

一款展示人类工具进化历程的合成游戏，玩家从原始自然资源开始，通过组合与工艺，逐步解锁从石器时代到量子时代的 110+ 种物品。

## ✨ 特性

- **8 个时代**：原始元素 → 石器 → 青铜 → 铁器 → 工业 → 信息 → 生物 → 量子
- **4 大学科**：物理、化学、生物、量子力学
- **16 种工艺**：手工、烧制、冶炼、锻造、退火、淬火、电解、光刻、发酵、培养、基因工程、核反应、量子工艺、超低温等
- **两个版本**：
  - 🎮 **简单版**：友好入门，流畅体验
  - 📚 **真实版**：万物说明书，含化学式、熔点、密度等真实参数
- **单机离线**：进度保存在浏览器 localStorage，无需服务器
- **跨平台**：PC、手机浏览器、Android APK（通过 Capacitor）
- **PWA 支持**：可"添加到主屏幕"作为离线应用

## 🎮 玩法

1. 点击基础资源放入合成台（最多 4 个材料 + 1 个工具）
2. 选择合适的工艺（不同工艺需要不同温度/电力）
3. 点击"合成"按钮，发现新物品
4. 解锁新物品后，可继续作为合成材料
5. 通过 8 个时代，最终目标：戴森球！

## 🚀 快速开始

### 单机模式（推荐）

直接在浏览器打开 `standalone.html` 即可，所有进度保存在本地。

```powershell
# 任选一种方式启动静态服务器
python -m http.server 8080
# 或
npx serve .
```

然后访问 http://localhost:8080/standalone.html

### 服务器模式（多人进度保存）

```powershell
npm install
node server.js
# 访问 http://localhost:8080
```

服务器模式支持：
- 用户注册/登录（bcryptjs 加密）
- 进度云端保存（SQL.js 数据库）
- 排行榜

### 公网访问（Cloudflare Tunnel）

```powershell
.\cloudflared.exe tunnel --url http://localhost:8080
```

会生成一个 `*.trycloudflare.com` 的临时公网地址。

## 🛠️ 技术栈

- **前端**：原生 HTML5 + CSS3 + JavaScript（无框架）
- **后端**（可选）：Node.js + Express + SQL.js + bcryptjs
- **PWA**：manifest.json + service worker
- **APK**：Capacitor
- **CI/CD**：GitHub Actions
